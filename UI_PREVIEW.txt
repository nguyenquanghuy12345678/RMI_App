FEATURES:

✓ Kết nối RMI Server từ xa
✓ Chuyển khoản real-time
✓ Kiểm tra số dư tự động
✓ Cập nhật đồng bộ 2 database
✓ Transaction-based (ACID)
✓ Auto rollback khi lỗi
✓ Giao diện thân thiện
✓ Tạo tài khoản mới
✓ Xem danh sách TK
